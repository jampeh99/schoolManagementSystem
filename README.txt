All 3 sections allow users to signup and login

		Pass credentials are stored in "admin.txt"

	2. Student Database
	Pass credentials are stored in "studentDatabase.txt"
	Allows students to;
		Add courses
		Edit Courses
		Delete Courses

	3. Staff Database
		Pass credentials are stored in "staffDatabase.txt"
